# نظام تحليل ركود المخزون المتقدم (Power BI Stagnation Analysis System)

## وصف المشروع
نظام متكامل تم تطويره باستخدام Power BI لتحليل حركة المخزون وتحديد المنتجات الراكدة بدقة عالية. يتميز النظام بقدرته على حساب "أيام كفاية المخزون" (Days to Stockout) ليس فقط للمنتجات التامة، بل وللمواد الخام أيضاً من خلال منطق "المخزون المكافئ".

## الميزات التقنية الرئيسية
*   **حساب أيام كفاية المخزون (Days to Stockout):** معادلات DAX متقدمة تتنبأ بالموعد المتوقع لنفاد المخزون بناءً على متوسط معدلات السحب اليومية.
*   **منطق المخزون المكافئ (Equivalent Stock Logic):** ربط المواد الخام بالمنتجات التامة باستخدام جدول (BOM - Bill of Materials)، مما يسمح بمعرفة مدى كفاية الخام بناءً على سحب المنتجات النهائية.
*   **مقياس التفسير النصي الذكي (Logic Details Measure):** مقياس يوفر شرحاً نصياً تلقائياً لكل صنف (مثال: "مخزون 500 ÷ سحب يومي 10 = يكفي لـ 50 يوم")، مما يسهل على متخذ القرار فهم حالة الصنف فوراً.
*   **معالجة المخزون الموحد:** حساب إجمالي المخزون على مستوى الصنف عبر كافة المستودعات باستخدام دالة `ALLEXCEPT` لتجاوز تشتت البيانات.
*   **تصنيف المنتجات:** تمييز تلقائي بين المنتجات التامة، المواد الخام، والمستهلكات مع تطبيق منطق حسابي خاص لكل نوع.

## الأدوات المستخدمة
*   **Power BI Desktop:** لنمذجة البيانات وبناء التقارير.
*   **DAX (Advanced):** لكتابة المقاييس المعقدة ومعالجة التبعيات الدائرية (Circular Dependencies).
*   **Excel:** كمصدر أساسي للبيانات (المخزون، المبيعات، ربط الأصناف).

## كيفية الاستخدام
1. قم بتحميل ملف `JOJO_New.pbix`.
2. افتح الملف باستخدام Power BI Desktop.
3. استعرض صفحة "تحليل الركود" لمشاهدة التصنيفات والتفاصيل النصية لكل منتج.
